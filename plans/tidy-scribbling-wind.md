# 게임 페이지 PC 우선 레이아웃 개선

## Context
현재 게임 페이지(`/game`)가 `max-w-lg`(512px) 단일 칼럼으로 모든 요소가 세로로 쌓여있음.
데스크탑에서 양쪽 여백이 과도하고, 스크롤이 길어져 스탯을 보면서 선택하기 불편함.
PC 우선 2컬럼 레이아웃으로 변경하여 공간 활용 최적화.

### Mobile (lg 미만): 현재와 동일한 세로 스택 유지

## 구현 계획

### 1. `src/app/game/page.tsx` — 2컬럼 레이아웃
- 전체 컨테이너: `max-w-lg` → `max-w-5xl` (desktop 넓게)
- `lg:grid lg:grid-cols-[320px_1fr] lg:gap-6` 2컬럼 그리드
- 왼쪽(사이드바): 헤더, WeekCounter, StatPanel, EmergencyButton → `lg:sticky lg:top-6`
- 오른쪽(메인): EventCard + ChoiceButton 목록
- 모바일: 기존처럼 `flex flex-col gap-4`

### 2. `src/components/game/StatPanel.tsx` — 반응형 그리드
- `grid-cols-2` → `grid-cols-2 lg:grid-cols-1` (사이드바에서 1컬럼 세로 리스트)

### 3. 기타 미세 조정
- 헤더 바: 닉네임+점수를 전체 너비 상단에 배치 (`col-span-full`)
- EmergencyButton: 모바일에서는 메인 영역 하단, 데스크탑에서는 사이드바 하단

## 수정 파일
- `src/app/game/page.tsx` (주요 변경)
- `src/components/game/StatPanel.tsx` (반응형 그리드 추가)

- 선택지 클릭 후 이벤트 전환이 정상 동작하는지 확인
