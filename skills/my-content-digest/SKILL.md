# My Content Digest

콘텐츠(트윗, YouTube, 아티클 등)를 받아서 Quiz-First 방식으로 소화하는 스킬.
요약을 먼저 보여주지 않고, 퀴즈부터 출제하여 학습 효과를 극대화한다.

> **Quiz-First 원리**: 학습 전 테스트(Pretesting Effect)가 기억력을 9-12% 향상시킨다.
> 틀린 문제가 호기심(Information Gap)을 만들고, 호기심이 기억을 강화한다.

## Execution Flow

```
┌─────────────────────────────────────────┐
│  1. 콘텐츠 받기 (fetch 스킬 결과물)       │
├─────────────────────────────────────────┤
│  2. Pre-Quiz (3문제)                     │
│     요약 보기 전에 퀴즈부터!              │
├─────────────────────────────────────────┤
│  3. 선택적 콘텐츠 제공                    │
│     틀린 부분만? 인사이트만? 전체 요약?    │
├─────────────────────────────────────────┤
│  4. 본 퀴즈 (9문제)                      │
│     기본 3 + 중급 3 + 심화 3             │
├─────────────────────────────────────────┤
│  5. 결과 + 후속 선택                      │
└─────────────────────────────────────────┘
```

## Step 1: Pre-Quiz (콘텐츠 보기 전 퀴즈)

**핵심 원칙: 요약을 먼저 보여주지 않는다.**

### 출제 프로세스

1. 콘텐츠에서 핵심 주제 3개를 추출한다
2. AskUserQuestion으로 3문제를 출제한다:
   - 문제 1: 핵심 개념 관련
   - 문제 2: 저자/발화자의 핵심 주장 관련
   - 문제 3: 주요 원칙/방법론 관련
3. 각 문제는 4지선다 (label + description 형식)

### 출제 원칙

- ✅ **출제 대상**: 핵심 메시지, 주요 원칙, 저자 핵심 주장, 핵심 개념
- ❌ **출제 금지**: 날짜, 통계 수치, 지엽적 세부사항, 코드 문법

### 피드백

각 문제에 대해 즉시 정답/오답 피드백을 제공한다:
- 정답: "맞습니다! [간단한 보충 설명]"
- 오답: "아쉽습니다. 정답은 [정답]입니다. 이 부분을 콘텐츠에서 확인해보세요."

## Step 2: 선택적 콘텐츠 제공

Pre-Quiz 결과를 바탕으로, 사용자가 원하는 깊이를 선택한다.

AskUserQuestion으로 아래 선택지를 제시한다:

```
어떤 방식으로 콘텐츠를 확인할까요?
  □ 틀린 문제 관련 섹션만 — 오답 부분의 답을 콘텐츠에서 찾기
  □ 핵심 인사이트 3개 — 가장 중요한 포인트만 빠르게
  □ 전체 요약 + 인사이트 — 종합적 분석
  □ 바로 본 퀴즈로 — 요약 없이 9문제 도전
```

### 각 선택지별 제공 내용

- **틀린 문제 관련 섹션만**: Pre-Quiz에서 틀린 문제와 관련된 콘텐츠 부분만 발췌하여 제공
- **핵심 인사이트 3개**: 콘텐츠의 가장 중요한 3가지 포인트를 간결하게 정리
- **전체 요약 + 인사이트**: 전체 요약(3-5문장) + 인사이트(3개) + 핵심 키워드
- **바로 본 퀴즈로**: 콘텐츠 확인 없이 바로 본 퀴즈 진행

## Step 3: 본 퀴즈 (9문제)

3단계 난이도 × 3문제 = 총 9문제로 구성한다.

### 기본 (3문제)

- 핵심 메시지 이해
- 주요 개념 파악
- 기본 용어/정의

### 중급 (3문제)

- 개념 간 관계 파악
- 근거와 주장 연결
- 원인과 결과 이해

### 심화 (3문제)

- 실제 상황에 적용
- 핵심 원리의 확장/응용
- 비판적 사고 (저자 주장의 한계, 반론)

### 출제 방식

- 각 단계별로 AskUserQuestion을 사용하여 3문제씩 출제
- 각 단계 완료 후 즉시 정답과 해설을 제공
- 출제 원칙: 핵심 주제만 다룬다 (지엽적 세부사항/날짜/통계 금지)

## Step 4: 결과 + 후속 선택

본 퀴즈 완료 후 결과를 정리하고, 후속 행동을 선택받는다.

### 결과 정리

```
═══════════════════════════════════
  Content Digest 결과
═══════════════════════════════════

  Pre-Quiz: 2/3 정답
  본 퀴즈:  7/9 정답 (기본 3/3, 중급 2/3, 심화 2/3)

  강점: [잘 이해한 영역]
  보완: [추가 학습이 필요한 영역]

═══════════════════════════════════
```

### 후속 선택

AskUserQuestion으로 다음 행동을 선택받는다:

```
다음에 어떻게 할까요?
  □ 다른 문제로 재퀴즈 — 틀린 영역 중심으로 새 문제
  □ 관련 콘텐츠 추천 — 이 주제를 더 깊게 다룬 콘텐츠 추천
  □ 종료 — 학습 완료
```
